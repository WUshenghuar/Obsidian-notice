## ⭐ 必装·几乎所有用户都在用

| 插件 | GitHub | 安装量 | 一句话 |
|------|--------|--------|--------|
| **Dataview** | `blacksmithgu/obsidian-dataview` | 社区#1 | 把 Obsidian 当数据库查，`FROM #tag WHERE score > 60` |
| **Templater** | `SilentVoid13/Templater` | 极高 | 超强模板引擎，支持 JS 脚本，替代核心模板功能 |
| **Obsidian Git** | `Vinzent03/obsidian-git` | 极高 | 自动 commit + push，vault 自动备份防丢 |
| **Quick Add** | `chhoumann/quickadd` | 极高 | 一键创建笔记/添加任务/运行脚本，捕获灵感最快 |
| **Calendar** | `liamcain/obsidian-calendar-plugin` | 极高 | 侧边栏日历，点哪跳哪，日记党必备 |

## 💡 你的场景最佳组合

```
📚 考试复习期 → Spaced Repetition + Kanban + Charts + Tracker
📝 笔记管理   → Dataview + Templater + Linter + Obsidian Git
✍️ 写文档     → Excalidraw + Mind Map + Banners
⚡ 平时捕获   → Quick Add + Calendar + Auto Link Title
```

---

> 安装方法：在 Obsidian 中打开 **设置 → 社区插件 → 浏览**，搜索插件名即可安装。
> 如果还没开启社区插件：**设置 → 第三方插件 → 关闭安全模式**。
